# Row config

## Properties

```tsx
export interface RowConfig { 
  height?: number;
  border?: boolean;
}
```

### height

| Type | Required | Default |
| ---- | -------- | ------- |
| number | false |  48 |

Configures the height of the row.

### border

| Type | Required | Default |
| ---- | -------- | ------- |
| boolean | false |  false |

If true, a border will be added to the bottom of the row.
