# Beast Configuration

Configure your grid with a simple `BeastGridConfig` object. check the [example](example/basic) for a complete example.


## Parameters

### theme

The theme name to use. The default theme is `default`.

### onSortChange

A callback function that is called when the sort changes. 
The function receives the sorted data and the sorted columns.

## Properties

```tsx
export interface SortConfig {
  enabled: boolean;
  multiple: boolean;
}
export interface StyleConfig {
  maxHeight: number;
  border: boolean;
}
export interface RowConfig { 
  height: number;
  border: boolean;
}
export interface HeaderConfig {
  height: number;
  border: boolean;
}
export interface dragOptions {
  autoScrollSpeed: number;
  autoScrollMargin: number;
}

export interface BeastGridConfig<T> extends Partial<TableStyles> {
  columnDefs: ColumnDef[];
  defaultColumnDef?: Partial<ColumnDef>;
  data: T;
  sort?: Partial<SortConfig>;
  row?: Partial<RowConfig>;
  header?: Partial<HeaderConfig>;
  style?: Partial<StyleConfig>;
  dragOptions?: Partial<dragOptions>;
}
```

### columnDefs

Array of column definitions. Check the [column definitions](columns/definitions) for more information.

### defaultColumnDef

Default [column definition](#columnDefs). Applied to all columns.

### data

Array of data.

### sort

Sort options. Check the [sort config](sort) for more information.

### row

Row options. Check the [row config](row) for more information.

### header

Header options. Check the [header config](header) for more information.

### style

Style options. Check the [style config](style) for more information.

### dragOptions

Drag options. Check the [drag options](drag) for more information.
