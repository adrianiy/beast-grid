# Sort config

## Properties

```tsx
export interface SortConfig {
  enabled: boolean;
  multiple: boolean;
}
```

### enabled

| Type      | Default value |
| --------- | ------------- |
| `boolean` | `false`       |

Enable or disable the sort feature.

### multiple

| Type      | Default value |
| --------- | ------------- |
| `boolean` | `false`       |

Enable or disable sorting on multiple columns.
