# Style config

## Properties

```tsx
export interface StyleConfig {
  maxHeight?: number;
  border?: boolean;
}
```

### maxHeight

| Type   | Required |
| ------ | -------- |
| number | false     |

If set, the height of the grid will be limited to this value.

### border

| Type    | Required |
| ------- | -------- |
| boolean | false     |

If set, the grid will have an outline border.
