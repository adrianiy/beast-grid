# Introduction to Beast Grid
 
Beast Grid is simple, powerful and flexible grid system for React.

It is built with focus on simplicity and performance. It is designed to be easy to use and easy to understand.


## What is Beast Grid?

A **simple**, **powerful** and **flexible** site generation framework with everything you love from Next.js.

## Core Features

- **Simple**: Beast grid is designed to be simple and easy to use. You will need to configure just a json and you are ready to go.
- **Powerful**: Beast grid is designed to be powerful using list virtualization and other performance optimizations.
- **Flexible**: Beast grid is designed to be flexible and can be used in any kind of project. Customize the look and feel with your own theme and tweak configuration options to make the grid fit your needs.
