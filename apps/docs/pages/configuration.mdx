# Beast Configuration

Configure your grid with a simple `BeastGridConfig` object. check the [example](example) for a complete example.



## Properties

```tsx
export interface SortConfig {
  enabled: boolean;
  multiple: boolean;
}
export interface StyleConfig {
  maxHeight: number;
  border: boolean;
}
export interface RowConfig { 
  height: number;
}
export interface HeaderConfig {
  height: number;
}

export interface BeastGridConfig extends Partial<TableStyles> {
  columnDefs: ColumnDef[];
  defaultColumnDef?: ColumnDef;
  data: TData[];
  sort?: Partial<SortConfig>;
  row?: Partial<RowConfig>;
  header?: Partial<HeaderConfig>;
  style?: Partial<StyleConfig>;
  dragOptions?: Partial<{
    autoScrollSpeed: number;
    autoScrollMargin: number;
  }>
}
```

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| [`columnDefs`](columns/definitions) | `ColumnDef[]` | `[]` | Array of column definitions |
| `defaultColumnDef` | `ColumnDef` | `undefined` | Default column definition. Applied to all columns |
| `data` | `TData[]` | `[]` | Array of data |
| `sort` | `Partial<SortConfig>` | `{ enabled: false, multiple: false }` | Sort options |
| `row` | `Partial<RowConfig>` | `{ height: 48 }` | Row options |
| `header` | `Partial<HeaderConfig>` | `{ height: 48 }` | Header options |
| `border` | `boolean` | `false` | Enable border |
| `dragOptions` | `Partial<{ autoScrollSpeed: number; autoScrollMargin: number; }>` | `undefined` | Drag options. autoScrollSpeed changes scroll speed when mouse is in edge zone (controlled with autoScrollMargin) |

## Parameters

### `theme`
#### Type: `string`

The theme name to use. The default theme is `default`.


### onSortChange
#### Type: (data: TData[], sortColumns: Column[]) => TData[];
#### Optional

A callback function that is called when the sort changes. 
The function receives the sorted data and the sorted columns.
